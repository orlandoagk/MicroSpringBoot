# AREP_MicroSpringBoot

## Ejecucion

1. `git clone `
2. `mvn compile`
3. `mvn clean package`
4. `java -cp target\classes co.edu.escuelaing.SpringBoot.MicroSpringBoot co.edu.escuelaing.SpringBoot.Controller`
